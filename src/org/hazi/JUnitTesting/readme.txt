-->If you are going to use the Junit for the first time then we need to add it to our project explorer. 

-->Right click on project, open new and then search for JUnit. Selec the latest version and then finish. Now you will be able to
use Junit.

-->Assert is a method useful in determining Pass or Fail status of a test case, The assert methods are provided by the class 
org.junit.Assert which extends java.lang.Object class.

-->List of JUnit asserts:
