-->Refer JarDemo and JarDemoTest projects for demo on packages tutorial and how to relate JAR file.

--> In order to create JAR file, you need to click on project and then right click --> Select export --> select appropriate project to be archived
and then click next.

--> It creates the required JAR file on specified path.

-->once jar file is created, you can add it to external/referenced libraries. for this, right click on project and then build path. Select
Add libraries to add JAR file. 

-->Now your main class can refer to jar file which is in external libraries.
